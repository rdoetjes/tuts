De shell
  De shell is de standaard UNIX commando interpreter, en alle commando's die we tot nu toe hehad hebben worden door de shell uitgevoerd. Het is echter mogelijk om met de file editor ("man vi" of anders "man ed") een eigen file aan te maken, en in deze file kun je dan een programma opbouwen in dezelfde syntax die je ook gewoon gebruikt, met uitbreiding van enkele loop structuren, zoals "while … do …" en "if … then …".
